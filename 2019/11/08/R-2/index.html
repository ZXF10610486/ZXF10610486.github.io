2.数据集

2.1数据集的概念

数据构成的矩形数组
行表示：观测；列表示：变量

2.2数据类型

标量、向量、矩阵、数组、数据框和列表。

2.2.1 向量

向量是c()用于存储：数字型、字符型、逻辑型数据的一维数组。

创建向量 – a <- c( )

创建标量 – a <- 常量

单个向量中的数据类型必须一致！！

访问向量中单个元素: a[ ];

访问向量中多个元素:a[c( )];

访问向量中从xx到xx的元素:a[ : ]

2.2.2矩阵

矩阵元素具有相同模式的二维数组

创建矩阵 – y <-matrix(vector, nrow = , ncol = , byrow = , dimnames = list( , ) )

vector代表存储元素的对象，nrow代表行数量，ncol代表列数量。

默认情况下byrow = FALSE(默认按列填充)，byrow = TRUE时矩阵元素按行填充。

dimnames用来给行和列的名称赋值（一般是字符型）。
